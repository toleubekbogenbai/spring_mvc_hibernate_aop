# CRUD приложение на Spring MVC с применением Hibernate и AOP
### Я использовал зависимости через XML file, базу данных mySQL, Spring MVC, view file JSP

--- 
* Technology stack:
    * Spring MVC
    * Hibernate
    * AOP
___

### Поля в базе данных (mySQL)

    * id (int)
    * surname (String)
    * name (String)
    * department (String)
    * salary (String)
